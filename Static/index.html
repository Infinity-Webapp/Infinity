INFINITY : A chatting platform for everyone

Infinity is an open source initiative.
Anyone can join infinity anytime, for free!
In today's world, privacy is the greatest concern in the mind of everyone online. Therefore our platform has a privacy mode which uses web sockets for chatting. Web sockets do not store even a fraction of your chat in any database. This mean that once the page is closed, the chat is lost, forever.
If you are a web devloper, then you can contribute to improve our UI on our github repository
